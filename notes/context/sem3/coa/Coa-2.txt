**BCD Code**

The BCD code is a way of representing decimal numbers using four binary digits (bits). Each digit in the decimal number is represented by a group of four bits, with the most significant bit (MSB) being the leftmost bit. The BCD code uses a specific pattern to represent each digit, where the MSB is always 1 and the remaining three bits represent the value of the digit.

**Conversion Formulas**

The formulas for converting decimal numbers to BCD are:

*   To convert a decimal number to BCD, multiply each digit by 10 raised to the power of its position, starting from the right (i.e., 0^0, 1^1, 2^2, etc.).
*   Add up the results to get the BCD representation.

For example, to convert the decimal number 123 to BCD:

*   1 × 10^2 = 100
*   2 × 10^1 = 20
*   3 × 10^0 = 3
*   Add up the results: 100 + 20 + 3 = 123

The BCD representation of the decimal number 123 is:

This represents the decimal number 123 in BCD format.
