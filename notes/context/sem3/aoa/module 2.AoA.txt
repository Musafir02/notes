This page outlines the binary search algorithm, a method for finding an element in a sorted list. The algorithm works by dividing the list in half and comparing the middle element to the target value. If the target is found, the algorithm returns its location. Otherwise, the process is repeated with the appropriate half of the list.

**Algorithm Overview**

The algorithm consists of four steps:

1. **Set Start and End**: Initialize two pointers, `start` and `end`, to the first and last elements of the list, respectively.
2. **Calculate Midpoint**: Calculate the midpoint of the range defined by `start` and `end`.
3. **Compare Midpoint to Target**: Compare the midpoint element to the target value. If they are equal, return the midpoint index.
4. **Update Range**: If the midpoint element is less than the target, update `start` to the midpoint index + 1. If it is greater, update `end` to the midpoint index - 1. Repeat steps 2-4 until the target is found or the range is empty.

**Example**

Suppose we have a sorted list `[1, 2, 3, 4, 5, 6, 7, 8, 9]` and we want to find the element `5`. We start by setting `start` to `0` and `end` to `8`. The midpoint is `4`, which is less than `5`. We update `start` to `5` and repeat the process. The new midpoint is `7`, which is greater than `5`. We update `end` to `6` and repeat the process. The new midpoint is `5`, which is equal to the target. We return the index `5`.

**Code**

Here is a simple implementation of the binary search algorithm in Python:
```python
def binary_search(arr, target):
    start = 0
    end = len(arr) - 1
    while start <= end:
        midpoint = (start + end) // 2
        if arr[midpoint] == target:
            return midpoint
        elif arr[midpoint] < target:
            start = midpoint + 1
        else:
            end = midpoint - 1
    return -1  # Target not found
```
This code uses a while loop to repeatedly divide the range in half until the target is found or the range is empty. The `midpoint` variable is calculated using the formula `(start + end) // 2`, which rounds down to the nearest integer. The `if` statement checks if the midpoint element is equal to the target, and if so, returns the midpoint index. If the midpoint element is less than the target, the `start` pointer is updated to the midpoint index + 1. If it is greater, the `end` pointer is updated to the midpoint index - 1. The loop continues until the target is found or the range is empty, at which point the function returns `-1` to indicate that the target was not found.
