**Logical Instructions**

Logical instructions are used to perform logical operations on data. There are two main types of logical instructions: bitwise AND, OR, XOR, NOT, and TEST.

*   **Bitwise AND**: This instruction performs a bitwise AND operation on two operands. It compares each bit of the first operand to the corresponding bit of the second operand. If both bits are 1, the resulting bit is set to 1. Otherwise, it is set to 0.
*   **Bitwise OR**: This instruction performs a bitwise OR operation on two operands. It compares each bit of the first operand to the corresponding bit of the second operand. If either bit is 1, the resulting bit is set to 1. Otherwise, it is set to 0.
*   **XOR**: This instruction performs a bitwise XOR operation on two operands. It compares each bit of the first operand to the corresponding bit of the second operand. If the bits are different, the resulting bit is set to 1. Otherwise, it is set to 0.
*   **NOT**: This instruction performs a bitwise NOT operation on a single operand. It flips all the bits of the operand.
*   **TEST**: This instruction tests whether a register is zero or not. If the register is zero, the instruction sets the condition code to 1. Otherwise, it sets the condition code to 0.

**Shift and Rotate Instructions**

Shift and rotate instructions are used to shift or rotate the bits of a register.

*   **SHL/SAL**: This instruction shifts the bits of the register left by a specified number of places. The leftmost bit is filled with zeros, and the rightmost bit is discarded.
*   **SHR**: This instruction shifts the bits of the register right by a specified number of places. The rightmost bit is filled with zeros, and the leftmost bit is discarded.
*   **SAR**: This instruction shifts the bits of the register right by a specified number of places. The rightmost bit is filled with the sign bit (i.e., the most significant bit), and the leftmost bit is discarded.
*   **ROL**: This instruction rotates the bits of the register left by a specified number of places. The leftmost bit is filled with the rightmost bit, and the rightmost bit is discarded.
*   **ROR**: This instruction rotates the bits of the register right by a specified number of places. The rightmost bit is filled with the leftmost bit, and the leftmost bit is discarded.
*   **RCL**: This instruction rotates the bits of the register left through carry by a specified number of places. The leftmost bit is filled with the rightmost bit, and the rightmost bit is filled with the carry bit.
*   **RCR**: This instruction rotates the bits of the register right through carry by a specified number of places. The rightmost bit is filled with the leftmost bit, and the leftmost bit is filled with the carry bit.
